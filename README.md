# Vasyworks Docker Environment

Vasyworks（賃貸管理システム）をローカル環境で瞬時に構築・実行するためのDocker統合環境です。

## 🚀 特徴
* **コマンド一発起動**: `docker compose up -d` を実行するだけで、データベースの構築から全システムの立ち上げまで全自動で完了します。
* **テストデータ自動投入**: 起動時にテストデータが自動的にデータベースへ流し込まれるため、インストール後すぐにデモ画面を確認可能です。

## 📋 システム構成
本環境は以下のプロジェクトで構成されています。
* **vacancy_model**: VasyworksDB（データベースの定義・管理）
* **vacancy_mgr**: VasyworksMGR（物件データの登録・管理）
* **vacancy**: VasyworksLIST（自社物件の空室情報の公開）
* **vacancy_api**: VasyworksAPI（物件データ取得用のAPI）
* **vacancy_search**: VasyworksSEARCH（物件データの検索）

## 🛠 クイックスタート

### 前提条件
* Docker Desktop がインストールされていること

### 手順
1. リポジトリをクローンします。
2. コマンドプロンプト（またはターミナル）でリポジトリのルートフォルダに移動します。
3. 以下のコマンドを実行してください。
   ```bash
   docker compose up -d --build
   ```
4. 起動後、各ポート（8000～8004）にアクセスしてください。
